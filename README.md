# 🏫 Preschool Management System

A backend web application built during my **3-month internship** using Java and Spring Boot.
This system manages preschool operations including student records, admissions, and staff data.

---

## 🛠️ Tech Stack

| Layer      | Technology                        |
|------------|-----------------------------------|
| Backend    | Java, Spring Boot, Spring MVC     |
| Database   | MySQL   |
| API        | REST API (JSON-based)             |
| Tools      | Postman, Maven, Git, GitHub       |

---

## ✨ Features

- 📋 Student registration and management
- 👩‍🏫 Staff/teacher record management
- 🔗 REST APIs for frontend-backend integration
- Database support (MySQL)

## 🚀 How to Run
```bash
# Clone the repository
git clone https://github.com/boradpriya0210/Preschool.git

# Navigate to project folder
cd Preschool

# Run the Spring Boot app
./mvnw spring-boot:run
```

App runs on: `http://localhost:8080`

---

## 📡 API Endpoints (Sample)

| Method | Endpoint              | Description            |
|--------|-----------------------|------------------------|
| GET    | /api/students         | Get all students       |
| POST   | /api/students         | Add new student        |
| PUT    | /api/students/{id}    | Update student         |
| DELETE | /api/students/{id}    | Delete student         |
