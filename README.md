# 🏫 Preschool Management System

A backend web application built during my **3-month internship** using Java and Spring Boot.
This system manages preschool operations including student records, admissions, and staff data.

---

## 🛠️ Tech Stack

| Layer      | Technology                        |
|------------|-----------------------------------|
| Backend    | Java, Spring Boot, Spring MVC     |
| Database   | MySQL   |
| API        | REST API (JSON-based)             |
| Tools      | Postman, Maven, Git, GitHub       |

---

## ✨ Features

- 📋 Student registration and management
- 👩‍🏫 Staff/teacher record management
- 🔗 REST APIs for frontend-backend integration
- Database support (MySQL)
